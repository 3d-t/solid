3D printing

To make a 3D printed version of your design is very easy. Just open the object and click on the icon in the viewer to turn off the grid and axis.

Then click on the export icon and select STL and save the file to your local device.

You can now import the STL file in a 3D printing application like Ultimaker Cura.

Below is the result of printing the object on an Creality 3D printer. As you notice the shape is not completely round because the cylinders are build up from facets. To make the rounding more smooth you can change the segments variable in the design from 40 to 200 for example.
